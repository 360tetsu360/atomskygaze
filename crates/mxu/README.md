s# MXU2.0
Make sure to use `-mmxu` `-mmxu2` supported gcc like [this](https://github.com/Dafang-Hacks/mips-gcc472-glibc216-64bit).

## _mx128_lu1q
Store memory as mxu vector.

## _mx128_sub_b
Subtract signed 16x8bit vector.

## _mx128_subuu_b
Subtract unsigned 16x8bit vector.

## _mx128_maxu_b
Max value.

## _mx128_minu_b
Min value.
